# Scenic

## Usage

```zsh
# for now
% cargo run -- example/slide.latex > dst/result.pdf
```

## Build

```zsh
% make build
```

## Test

```zsh
% make test
```

## Requirements

* [Tectonic](https://github.com/tectonic-typesetting/tectonic)
* [DejaVu](https://dejavu-fonts.github.io)
* [TeX Gyre Pagella](
https://www.gust.org.pl/projecets/e-foundry/tex-gyre/pagella)
